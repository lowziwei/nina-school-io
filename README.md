## How to Use This Course

**Daily commitment:** 3 hours/day, 7 days/week (21 hours/week)

Each day is split into:
- **Session 1 (~1.5h):** Deep study — lecture notes, derivations, or a primary paper
- **Session 2 (~1.5h):** Active work — reading, exercises, or review/synthesis

**Tools required:**
- Stata (IC license sufficient; ~$48 USD for 6 months from stata.com)
- Python with [pyBLP](https://pyblp.readthedocs.io/) for Weeks 3–5 (optional but recommended)
- LaTeX for problem set write-ups (optional)

**Recommended background:**
- Intermediate microeconomics (indifference curves, profit maximization, game theory basics)
- Econometrics (OLS, IV, panel data)
- Probability and statistics
